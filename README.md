# 🔍 密评报告附录D 对比工具

> 上传两份密评报告（docx），自动解析附录D，**秒级生成HTML差异对比结果**

![Python](https://img.shields.io/badge/Python-3.8+-blue.svg)
![Flask](https://img.shields.io/badge/Flask-Web-green.svg)
![License](https://img.shields.io/badge/License-MIT-orange.svg)

## ✨ 功能特性

| 功能 | 说明 |
|------|------|
| 📄 **双文件上传** | 拖拽或选择任意两份密评docx报告 |
| ⚡ **即时HTML结果** | 解析完成即展示，无需等待Word/PDF生成 |
| 📊 **D.x.y.z层级结构** | 按原报告格式逐层展开：`D.4 安全计算环境 → a) 堡垒机 → 表格` |
| 🎨 **颜色差异标注** | 🔴降级 / 🟢升级 / 🟠变不适用 / 🔵从不适用变具体评价 / 🗑️删除 / ➕新增 |
| 🔍 **智能搜索过滤** | 关键词搜索 + 按章节/差异类型筛选 + "仅显示有差异项" |
| 🖨️ **打印导出PDF** | 浏览器一键打印或"另存为PDF" |
| 📱 **响应式设计** | 支持桌面和移动端查看 |

## 🚀 快速开始

### 方式一：Windows 一键启动（推荐）

```bash
# 双击 start.bat 即可
# 或在命令行中：
cd compare_tool
start.bat
```

### 方式二：手动启动

```bash
# 安装依赖（首次）
pip install flask

# 启动服务
python app.py

# 浏览器自动打开 http://127.0.0.1:5678
```

### 方式三：指定端口

```bash
python app.py --port 8080
```

## 📋 差异分类说明

| 分类 | 含义 | 示例 | 颜色 |
|------|------|------|------|
| **⬇️ 降级** | 符合程度变差 | `符合` → `不符合` | 🔴 红色 |
| **⬆️ 升级** | 符合程度变好 | `不适用` → `符合` | 🟢 绿色 |
| **→ 不适用** | 变为不适用 | `符合` → `不适用` | 🟠 橙色 |
| **← 从不适用变为具体评价** | 原本跳过的项目终稿做了核查 | `不适用` → `部分符合` | 🔵 蓝色 |
| **🗑️ 仅初稿有** | 终稿中删除该项 | — | 灰色+删除线 |
| **➕ 仅终稿有** | 终稿新增该项 | — | 绿色背景 |

## 📝 支持的报告格式

工具专门针对**密评报告附录D**格式优化，支持：

- ✅ GB/T 22239-2020 密评标准报告
- ✅ 包含"单项测评结果记录"的docx文档
- ✅ 表头包含"控制点"、"指标/要求"、"符合程度"列
- ✅ D.1~D.10 十大安全类层级结构

## ⚙️ 自定义配置

修改 `app.py` 中的配置：

```python
app.config['MAX_CONTENT_LENGTH'] = 50 * 1024 * 1024  # 最大文件大小: 50MB
# 修改端口号:
port = 5678  # 默认端口
host = '127.0.0.1'  # 监听地址
```
